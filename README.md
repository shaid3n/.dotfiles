# .dotfiles
neovim + tmux workflow

![Screenshot1](https://i.imgur.com/LJ7ITKh.png) 

## Install
```bash
git clone https://github.com/GatinSDK/.dotfiles.git
cd .dotfiles
stow .
```

## Packages

Arch

```bash
sudo pacman -S neovim tmux fish starship zoxide fzf stow ripgrep git sed unzip base-devel openssh lazygit python3 tree-sitter-cli
```
Others

``
fnm yay sesh uv bun cuda opencl-mesa
``
